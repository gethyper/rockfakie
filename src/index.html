Android prototypes in minutes!

Rapid Android Prototyping was created for super fast and easy Android prototyping. It's based on the Android Styleguide to help you create fully functional prototypes in minutes.


Easy to use

Rapid Android Prototyping allows us to create Android prototypes using basic HTML, CSS (LESS) and JavaScript. Everything is done in one single HTML file that can be easily host anywhere.

You can pretty much copy and paste snippets.


Native Look

Rapid Android Prototyping is based of the Android Style guide, that means all components have the native Android look and feel. That doesn't mean you cannot extend it. We use CSS variables that allow you to change the basic theme colors and of course you can create your own style with CSS.


Nice animations

With CSS3 it's possible to create smooth animations for mobile devices. Rapid Android Prototypinghas 4 types of transitions: Normal, ZOOM, Slide and Slide 3D.

You can also create your own transitions using Javascript and CSS3
